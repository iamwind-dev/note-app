# Note App - Flutter Provider State Management

A complete Flutter note-taking application built with Provider for state management. This app demonstrates how to manage app-wide state using `ChangeNotifier` and `Provider`.

## Features

- ✅ Create, edit, and delete notes
- ✅ Global state management using Provider
- ✅ Real-time UI updates when notes change
- ✅ Confirmation dialog before deletion
- ✅ SnackBar messages for user feedback
- ✅ Beautiful UI with Google Fonts (Poppins)
- ✅ Responsive card-based design
- ✅ Light/Dark theme support
- ✅ Empty state with helpful message

## How to Run

1. **Install dependencies:**

   ```bash
   flutter pub get
   ```
2. **Run the app:**

   ```bash
   flutter run
   ```
3. **Hot reload support:**

   - Press `r` for hot reload
   - Press `R` for hot restart
